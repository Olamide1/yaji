# Yaji Lagos.

To run;

for backend:
```cmd
> yarn|npm run start
```

> Open localhost:3000

 To deploy to heroku:

 ```cmd
 git push heroku main
 ```

 To see heroku logs
 ```cmd
 heroku logs -n 1500 --app sth-here >> heroku.logs
 ```
